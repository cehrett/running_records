# Functional and Non-Functional Requirements

## Functional Requirements

* The user should be able to upload audio files of the child reading the ground truth text.
* The user should be able to see the estimated score of the Running Record.
* The user should be able to get the indexes of repetitions, substitutions and insertions.

## Non-Functional Requirements

* The system should accurately transcribe audio of the reading.
* The system should have as few demographic blind spots as possible.
* The system should be effective for different ground truth texts.
* The system's Watson Transcript should include all repetitions, substitutions and insertions.
* The system should run on Windows, Mac and Ubuntu.
